ТЕМА РОБОТИ:

КАСКАДНІ ТАБЛИЦІ СТИЛІВ. СЕЛЕКТОРИ .ІДЕНТИФІКАТОРИ. СТИЛЬОВЕ ОФОРМЛЕННЯ ТЕКСТОВИХ ЕЛЕМЕНТІВ В HTML- ДОКУМЕНТАХ.

МЕТА:

ПРИДБАТИ ПРАКТИЧНІ НАВИЧКИ РОБОТИ ЗІ СТИЛЬОВИМ ОФОРМЛЕННЯМ HTML-ДОКУМЕНТА

МІСЦЕ РОЗТАШУВАННЯ:

РЕПОЗИТОРІЙ НА GITHUB

Селектор тегу:

h1, h2, h3, h4, p, ul { margin: 0; padding: 0; }

Селектор класу:

.container-header{ display: flex; align-items: center; }

Ідентифікатор:

<a href="./index.html" class="logo"> Happy<span class="logo decor" id="logo-web">Studio</span> </a>

Універсальний селектор:

*.warning { color: red; }

Дочірні селектори:

.section > item{ width: 270px; }

Оформлення списків:

.social{ display:flex; justify-content: center; align-content: center; margin-bottom: 30px; }

Відступи:

.hero-title{ max-width: 696px; margin-bottom: 30px; margin-left: auto; margin-right: auto; }

Оформлення фону:

.hero-content{ width: 1600px; margin-left: auto; margin-right: auto; padding-top: 200px; padding-bottom:200px; text-align: center; background-color: var(--bgcolor-footer); background-image: linear-gradient(rgba(47, 48, 58, 0.4), rgba(47, 48, 58, 0.4)), url(../images/herobg.jpg); background-repeat: no-repeat; background-position: center; }

ВИСНОВКИ:

У ході виконання лабораторної роботи з CSS ми здобули практичні навички створення стильового оформлення веб-сторінок, ознайомились з різними видами селекторів та властивостей і навчились застосовувати каскадні таблиці стилів для ефективного контролю над візуальним представленням HTML-документів. Робота зі стилями дозволила нам краще зрозуміти, як домогтися бажаної візуалізації веб-елементів, враховуючи адаптивність та кросбраузерну сумісність, підкресливши значення глибокого розуміння принципів дизайну та розробки сучасних веб-ресурсів.
